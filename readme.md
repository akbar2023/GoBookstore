# Few important commands

### Import gin library
```go get github.com/gin-gonic/gin```

### Project init
```go mod init example/web-service-gin```

### Import gorm (ORM for Go projetcs)
```go get -u gorm.io/gorm```

### Import Mysql driver
```go get -u gorm.io/driver/mysql```

### Build project with exec
```go build```
